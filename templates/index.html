About QGIS API Documentation

This site provides API documentation for QGIS, the free and open-source geographic information system. The documentation is generated using Doxygen from the QGIS C++ source code.

Choose a version above to browse the API reference. For most users, we recommend using the latest LTS release for stable development, or master for bleeding-edge features.
